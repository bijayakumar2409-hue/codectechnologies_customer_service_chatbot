# 🤖 Customer Service Chatbot

An AI-powered Customer Service Chatbot developed as part of the CodeTech Technologies Internship. The chatbot helps users by answering common customer queries through an interactive interface.

## 🚀 Features
- Customer query handling
- Interactive chatbot interface
- Fast response
- Simple and user-friendly UI

## 🛠️ Built With
- Python
- Flask
- HTML5
- CSS3
- JavaScript

## 📁 Project Structure

customer_service_chatbot/
├── app.py
├── templates/
│   └── index.html
├── requirements.txt
└── README.md

## ▶️ How to Run

1. Install requirements:
   pip install -r requirements.txt
2. Run:
   python app.py
3. Open:
   http://127.0.0.1:5000
